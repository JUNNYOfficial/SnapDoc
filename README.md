# SnapDoc

一款专注于沉浸式长文写作的浏览器端编辑器。没有标签页、没有复杂排版，只有一个无限滚动的编辑空间，和一套为"写到一半"设计的缓冲机制。

## 核心特性

### 单流编辑体验
- **无限滚动画布** — 只有一个连续的编辑区域，没有分页、没有标签干扰
- **块级结构** — 支持标题、正文、引用、列表等多种内容块，回车自动创建新块
- **实时字数统计** — 底部状态栏实时显示字符数、阅读时长估算

### 文档管理
- **本地存储** — 所有文档保存在浏览器 localStorage，无需后端
- **多文档切换** — 侧边栏文档列表，支持重命名、删除
- **历史版本** — 自动保存版本记录（每5分钟或内容变化超100字符），支持一键恢复到任意版本
- **版本压缩** — 智能压缩中间版本，保留首尾版本并生成变化摘要，节省存储空间

### 文档模板
- **6种预设模板**：空白文档、每日日记、周报、项目计划、会议记录、读书笔记
- 一键创建，自动填充结构化内容

### 导入与导出
- **导入**：支持 Markdown (.md)、纯文本 (.txt)、Word (.docx via mammoth)、PDF (文字提取)
- **导出**：支持 Markdown、PDF (浏览器打印)、HTML 分享链接
- **分享功能** — 生成独立 HTML 文件，对方可在极简阅读界面查看

### 写作辅助
- **语音草稿** — 浏览器语音识别，边说边写
- **拼写检查** — 英文常见错误检测、中文重复词检测
- **搜索替换** — 支持正则表达式
- **大纲视图** — 自动提取标题生成文档大纲
- **分屏预览** — 实时 Markdown 预览

### 主题与视觉
- **4种主题**：深空黑、深青、羊皮纸、午夜蓝
- **纸质纹理背景** — 可选纸张噪点质感
- **全 SVG 图标** — 无 emoji，统一视觉风格

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + S` | 保存文档 |
| `Ctrl/Cmd + F` | 搜索 |
| `Ctrl/Cmd + B` | 加粗 |
| `Ctrl/Cmd + I` | 斜体 |
| `Ctrl/Cmd + K` | 插入链接 |
| `Ctrl/Cmd + /` | 快捷键帮助 |
| `Ctrl/Cmd + Shift + V` | 粘贴为纯文本 |

## 技术栈

- **纯前端实现** — 单 HTML 文件，零构建步骤
- **localStorage 持久化** — 文档、版本、设置均本地存储
- **PDF.js** — PDF 文字提取与解析
- **mammoth.js** — Word 文档导入

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/JUNNYOfficial/SnapDoc.git
cd SnapDoc
```

2. 启动本地服务器
```bash
./run.sh
# 或
python3 -m http.server 8080
```

3. 浏览器打开 `http://localhost:8080`

## 浏览器兼容性

- Chrome / Edge / Safari / Firefox 最新版
- 需要支持 `contenteditable`、`localStorage`、`Web Speech API`（语音功能）
